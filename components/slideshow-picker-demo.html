Slideshow Picker Component

Reusable picker UI for product slideshows, image testing, and background switching

Product Slideshow

Interactive background slideshow with overlay controls. User's preferred images: Desert, Mountain, Workspace (all with light overlay).

Your Product Here

This slideshow picker can be used for hero sections, product showcases, or any visual content that benefits from multiple background options.

// Initialize the slideshow picker
const picker = new SlideshowPicker({
  container: '#picker-embedded',
  target: '#slideshow-target',
  slides: [
    { id: 'desert', label: 'Desert', image: 'desert-sunset.jpg' },
    { id: 'mountain', label: 'Mountain', image: 'mountain-milkyway.jpg', active: true },
    { id: 'workspace', label: 'Workspace', image: 'workspace.jpg' },
  ],
  controls: [
    { id: 'light', label: 'Light', active: true },
    { id: 'standard', label: 'Standard' },
    { id: 'heavy', label: 'Heavy' },
  ],
  showInfo: false
});

Component Files

Include these files to use the slideshow picker:

  • slideshow-picker.css - Styles for picker UI (dark/light themes, sizes, positions)
  • slideshow-picker.js - JavaScript class for picker functionality

Options

Available configuration options:

Option Type Default Description
container string/element null Selector or element for picker container
target string/element null Element to apply background classes to
slides array [] Array of {id, label, image, active}
controls array [] Array of {id, label, active}
theme string 'dark' 'dark' or 'light'
size string 'default' 'small', 'default', or 'large'
autoplay boolean false Enable auto-advance
autoplayInterval number 5000 Ms between slides
